## 基于React+antd实现后台模板
在某个github同学的仓库借鉴了她的模板，然后在上面做大幅度改动用来实现自己的功能。
这位github同学的项目非常适合初学者，她/他的github地址如下：

github地址：[基于React+antd实现后台模板](https://github.com/z-9527/react-admin-master)  
预览地址：[预览地址](https://z-9527.github.io/react-admin-master/)
